# What is the Cosmos SDK?

{: .no_toc }

Cosmos SDK 是一个开源框架，用于构建多资产的公共权益证明（Proof-of-Stake, PoS）区块链，如 Cosmos Hub，以及许可的权威证明（Proof-of-Authority, PoA）区块链。使用 Cosmos SDK 构建的区块链通常被称为特定应用的区块链。

Cosmos SDK 的目标是让开发者能够轻松地从零开始创建定制的区块链，这些区块链可以与其他区块链原生地互操作。我们将 Cosmos SDK 视为一个类似 npm 的框架，用于在 CometBFT 之上构建安全的区块链应用。基于 SDK 的区块链是由可组合模块构建的，其中大多数模块是开源的，任何开发者都可以随意使用。任何人都可以为 Cosmos SDK 创建一个模块，而且集成已经构建的模块就像将它们导入到你的区块链应用中一样简单。更重要的是，Cosmos SDK 是一个基于能力的系统，它允许开发者更好地理解模块间交互的安全性。要更深入地了解能力，请跳转到对象能力模型。

# 什么是特定应用的区块链

在当今区块链世界中，一个发展范式是像以太坊这样的虚拟机区块链，开发通常围绕在现有区块链之上构建去中心化应用作为一套智能合约。虽然智能合约对于某些用例（例如 ICO）可能非常有效，但它们在构建复杂的去中心化平台时往往不足。更一般地说，智能合约在灵活性、主权和性能方面可能具有限制性。

特定应用的区块链提供了与虚拟机区块链截然不同的开发范式。特定应用的区块链是为运行单一应用而定制的区块链：开发者拥有做出应用运行最佳所需的设计决策的全部自由。它们还可以提供更好的主权、安全性和性能。

# 为什么选择 Cosmos SDK
Cosmos SDK 是目前构建定制特定应用区块链的最先进框架。以下是一些你可能会考虑使用 Cosmos SDK 构建你的去中心化应用的原因：

* Cosmos SDK 内置的默认共识引擎是 CometBFT。CometBFT 是目前存在的最成熟（也是唯一的）拜占庭容错（BFT）共识引擎。它在整个行业中被广泛使用，并被视为构建权益证明系统的黄金标准共识引擎。
* Cosmos SDK 是开源的，并且旨在使构建由可组合模块组成的区块链变得容易。随着开源 Cosmos SDK 模块生态系统的增长，使用它构建复杂的去中心化平台将变得越来越容易。
* Cosmos SDK 的灵感来自基于能力的安全性，并且是多年来与区块链状态机斗争的成果。这使 Cosmos SDK 成为构建区块链的非常安全的环境。
* 最重要的是，Cosmos SDK 已经被用于构建许多已经投入生产的特定应用区块链。其中包括 Cosmos Hub、IRIS Hub、Binance Chain、Terra 或 Kava 等。还有更多正在使用 Cosmos SDK 构建。
